# Archive 目录说明

此目录包含项目开发过程中产生的临时文件、测试脚本、编译产物和备份文件。

## 文件分类

### 测试和诊断脚本
- `test_*.py` - 各种测试脚本
- `diagnose_*.py` - 诊断脚本
- `check_*.py` - 检查脚本
- `debug_*.py` - 调试脚本
- `e2e_*.py` - 端到端测试
- `自动化*.py` - 自动化测试脚本
- `详细检查并打包.py` - 打包脚本

### 批处理文件
- `*.bat` - 各种批处理启动和测试脚本

### 编译产物
- `CudaRuntime1.*` - CUDA 项目文件
- `x64/` - 编译输出目录
- `*.exe` - 可执行文件

### 备份和临时文件
- `*.zip` - 打包文件
- `*.log` - 日志文件
- `*.png` - 图片文件
- `html_files_20260206_150438/` - HTML 文件备份

### 文档
- `SUMMARY.md` - 项目总结
- `Dashboard数据说明.md` - 数据说明文档
- `完整验证流程说明.md` - 验证流程文档
- 其他中文文档

### 其他
- `cloudflared.exe` - Cloudflare 隧道工具
- `CNAME` - DNS 配置文件
- `config.yml` - 配置文件（已移动到根目录）

## 注意

这些文件已从项目根目录移至此目录，不影响项目的正常运行。
如需使用这些文件，请从 archive 目录中查找。
